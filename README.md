# 🔐 Password Generator (Day 5)

A simple Python-based password generator that creates secure passwords using a mix of letters, numbers, and symbols.

## 🚀 About the Project
This project is part of my **100 Days of Code** challenge.  
It demonstrates how to generate random passwords based on user-defined criteria.

## 🎯 Features
- User chooses:
  - Number of letters
  - Number of symbols
  - Number of numbers
- Randomly generates a password
- Shuffles characters for better randomness

## 🧠 Concepts Used
- Python input/output  
- Lists and string manipulation  
- Loops (`for`)  
- Random module (`random.choice`, `random.shuffle`)  

## ▶️ How to Run
1. Make sure Python is installed  
2. Run the script:
   ```bash
   python password_generator.py
